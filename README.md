# Gym Member Management Application

A front-end web application built with JavaScript, HTML, and CSS that allows users to manage gym member records through an interactive interface. This project simulates a real-world member management system with filtering, validation, and dynamic data handling.

---

## 🚀 Features

### Member Management
- Add new gym members with detailed information
- Edit existing member records
- Delete members from the system
- Automatically generate unique member IDs

### Data Handling
- Dynamic rendering of member list
- Real-time updates when adding/editing/removing members
- Structured data management using JavaScript objects/arrays

### Filtering & Search
- Filter by:
  - Member ID
  - Username
  - First Name
  - Last Name
- Toggle filters for:
  - Pool access
  - Spa access

### Validation
- Username validation (2 digits + 4–6 letters)
- Required field enforcement
- Date input validation (enrollment & cancellation)

### UI / UX
- Clean, responsive layout
- Toggle between:
  - Add/Edit Member view
  - Member List view
- Interactive buttons (Edit, Delete, Save, Cancel)
